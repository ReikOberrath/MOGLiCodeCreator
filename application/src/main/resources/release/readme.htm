The MOGLi Code Creator

 

Just another code generator? – NO! – It’s THE Lightweight autogeneration tool!

It’s written in Java but made to generate more than only Java code.

 

MOGLi stands for the following attributes:

 

M odel based

O pen for extension

G enerator based

Li ghtweight

 

Getting started

1.    Have a Java 1.6 runtime environment (JRE) installed.

2.    Provide this JRE to MOGLiCC

a)      either by setting the JAVA_HOME system variable

b)      or by editing the start batch file ‘startMOGLiCodeCreator.bat’ and setting the classpath variable.

3.  Start the batch file ‘startMOGLiCodeCreator.bat’. This will create a number of directories and files, for instance, an input directory that provides example input data, and a output directory with the corresponding result files.

4.  Having done so successfully, you find detailed information about MOGLiCC here: help/MOGLi Code Creator.htm.
