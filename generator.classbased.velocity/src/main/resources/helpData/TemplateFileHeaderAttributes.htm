The VelocityClassBasedGenerator plugin defines the following attributes which you can set for your purposes in a template header:

1.    TargetDir                                      Folder in your file system where to write the generated content.

2.    TargetFileName                        Name of file in which the generated content is saved.

3.    CreateNew                                 If “true” both the target folder is created if not existing and the target file is overwritten if existing. Default is false.

4.    NameOfValidModel                Determines that the target file is only generated if the name of the model provided by the StandardModelProvider is equal to the model name defined here.

By default the template is applied to all models.
