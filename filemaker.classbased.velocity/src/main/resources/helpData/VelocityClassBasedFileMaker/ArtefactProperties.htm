The VelocityClassBasedFileMaker plugin defines the following attributes which you can set for your purposes in a template header:

1.    TargetDir                                      Folder in your file system where to write the generated content.

2.    TargetFileName                        Name of file in which the generated content is saved.

3.    CreateNew                                 If “true” both the target folder is created if not existing and the target file is overwritten if existing. Default is false.

4.    NameOfValidModel                Determines that the target file is only modified if the name of the model provided by the StandardModelProvider is equal to the model name defined here. If not provided, the template is applied to all models.

5.   OutputEncodingFormat         Encoding applied when generating the artefact’s output file(s). Default is UTF-8. Since 1.1.0

6. SkipGeneration                          If “true”, code supposed to be created is not generated. Default is false. Used to suppress generation of files for certain classes, e.g. @SkipClass NOT $classDescriptor.getMetaInfoValueFor("databaseRelevant"). Since 1.2.0
